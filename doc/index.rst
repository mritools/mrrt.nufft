Welcome to pyir.nufft's documentation!
======================================

`pyir.nufft` is a package for computing non-uniform fast Fourier transforms
(NUFFT) as used in MRI.  Multithreaded CPU implementations exist in this
package.  A Non-Cartesian MRI system operator including coil sensitivities and
field maps is available in pyir.operators_private.  A GPU implementation is
available via pyir.cuda.

For more information, please refer to the readme_ in the Github repository.
